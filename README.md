# Quick Commerce Data Analysis Project

# Overview

This project simulates a quick commerce platform like Zepto and analyzes customer orders, delivery performance, and product sales using SQL.

# Database Design
The database includes:
* Customers
* Orders
* Order Items
* Products
* Delivery Partners
* Warehouses

# Key Analysis Performed

* Total Revenue Calculation
* City-wise Order Analysis
* Top Selling Products
* Delivery Time Analysis
* Peak Order Hours
* Repeat Customers

# Tools Used

* SQL (MySQL)

# Key Insights

* Identified top-performing products and categories
* Analyzed delivery efficiency
* Found peak ordering hours
* Identified loyal customers

# Files Included

* schema.sql → database structure
* data.sql → sample data
* queries.sql → analysis queries
